Data Collected

Customer data is collected during the order process:

Purpose of Data Processing

  • Customer data is used only for processing and delivering orders.
  • Chef login is used only for securely managing orders.

Data Storage

  • Customer data is stored in Firebase Firestore.
  • Login credentials are stored securely in Firebase Authentication.
  • Data is retained only as long as necessary to process the order.

Data Protection

  • Data is protected by Firebase security rules.
  • Data transmission occurs over secure HTTPS connections.
  • Access to data is limited to authorized personnel (chef).

Rights of the Data Subject

Customers have the right to:

  • Access their personal data
  • Request correction of their data

Consent
